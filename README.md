1. Object-Oriented Programming (Inheritance)
The repository includes practical examples of the four major types of inheritance in Python:
Single Inheritance: Basic Parent-Child relationship (e.g., Employee -> Manager).
Multilevel Inheritance: Chain of inheritance (e.g., Person -> Student -> Graduate).
Multiple Inheritance: A child class inheriting from multiple parent classes (e.g., Academic & Cocoricular -> Student).
Hierarchical Inheritance: Multiple child classes inheriting from a single parent (e.g., Person -> Undergraduate & Graduate).
